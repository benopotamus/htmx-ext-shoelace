Form submission rules

	
    
		
  1. Fields are only included if they have a name attribute.
    2. 
		
  2. Two included fields with the same name will result in 2 entries in that same name in the form submission.
    3. 
		
  3. Blank Input and TextArea elements are included
    4. 
		
  4. Select element value will be the value of the selected option, or if no value attribute is present, the innerText of the selected option element.
    5. 
		
  5. Unchecked checkboxes are not included
    6. 
		
  6. A checkbox without a value attribute will have a value of "on"
    7. 
		
  7. Unselected radio buttons are not included
    8. 
		
  8. A radio button without a value attribute will have a value of "on"
    9. 
		
  9. The select multiple attribute will include multiple entries with the same name, and their respective values.
